Project Information

This project USB Watch Dog, was personally developed by Dilip Atchuth Kumar Pulamarasetty as an independent cybersecurity project. It aims to enhance system security by implementing physical access control over USB ports, helping users prevent unauthorized data access and malware threats from removable devices.

Project Details

Project Name USB Watch Dog
Project Description A personal cybersecurity tool designed to enable or disable USB ports using a password-protected GUI, preventing unauthorized device access.
